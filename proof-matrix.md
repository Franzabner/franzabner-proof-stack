## Proof Row Gate

Every future proof row must include:

- artifact location;
- status label;
- evidence summary;
- excluded private or sealed material;
- validation method;
- review requirement;
- review date or pending review state;
- next action.

## Phase 1 Hold

Supporting narrow repos remain source context until separate review. No supporting repo patch, GitHub metadata change, Hugging Face metadata change, archive, transfer, rename, visibility change, release, deployment, certification, production CAD claim, active service claim, autonomous production-agent claim, or proof-completion claim is authorized by this matrix.
